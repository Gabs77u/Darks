1. **Clone o repositório:**
   ```bash
   git clone <url-do-repo>
   ```
2. **Crie e ative o ambiente virtual:**
   ```bash
   python -m venv venv
   source venv/Scripts/activate  # Windows
   ```
3. **Instale as dependências:**
   ```bash
   pip install -r requirements.txt
   ```
4. **Execute a aplicação:**
   ```bash
   python run_gui.py
   ```
